# Circom folder

Put here circom files, the `output/`:

```sh
circom multiply.circom --r1cs --wasm --sym --c -o output
circom add.circom --r1cs --wasm --sym --c -o output
circom deposit.circom --r1cs --wasm --sym --c -o output
```
